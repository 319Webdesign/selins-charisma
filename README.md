# Selin's Charisma – Friseursalon Website

Moderne, responsive Website für den Friseursalon Selin's Charisma, erstellt mit Next.js 14 und Tailwind CSS.

## Design

- **Farbschema:** Dark-Mode mit Deep Charcoal (#1a1a1a) und Gold-Akzenten (#D4AF37)
- **Schriften:** Playfair Display (Headlines), Inter (Fließtext)
- **Heller Bereich:** Beratung & Hypnose nutzt Cremeton (#f9f9f9)

## Technologien

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Lucide React (Icons)

## Entwicklung

```bash
npm install
npm run dev
```

Öffnen Sie [http://localhost:3000](http://localhost:3000) im Browser.

## Build

```bash
npm run build
npm start
```

## Seitenstruktur

- **/** – Home mit Hero, Leistungen (Damen/Herren/Kosmetik), Instagram-CTA
- **/preise** – Preisübersicht für Damen, Herren, Kosmetik
- **/beratung-hypnose** – Beratung & Hypnose mit hellem, vertrauensvollem Layout

## Platzhalter

Alle Medienbereiche nutzen Platzhalter-Komponenten (graue Boxen mit Label). Ersetzen Sie diese durch echte Bilder und Videos für das finale Shooting.
